# Module 1 : Web Frontend (Interface & Interaction)

1. Maquetter des Interfaces (Design)
    - Figma : Apprendre à concevoir pour le web
    - Création de Wireframes (structure)
    - Passage à la Maquette haute fidélité (UI)
    - Prototype interactif (flux de navigation)
    - Git : Export des assets et de la documentation de design

2. Développer l'Interface (Statique)
    - HTML5 & CSS3 : Sémantique, Flexbox et CSS Grid
    - Docker
        - Conteneur Nginx ou Apache pour servir les fichiers
        - Volumes pour voir les changements en temps réel
    - Git : Un dossier par projet/exercice

3. Gérer les Données & Interaction dynamique
    - JavaScript Vanilla : Manipulation du DOM, événements
    - Consommation de données (Fetch)
        - Utilisation de fichiers JSON statiques en lecture seule pour alimenter le DOM
    - Validation de formulaires
    - LocalStorage : Persistance de données côté client

4. Framework Frontend (VueJS ou React)
    - Composants : Découpage de l'interface en briques réutilisables
    - Props & State : Faire circuler les données entre les composants
    - Docker : Création d'image personnalisée

5. Déploiement & Documentation
    - Markdown : Rédaction du README.md expliquant comment lancer le projet avec Docker
    - CI/CD Simplifié : Introduction au déploiement via Git
